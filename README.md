# NFL Quarterback GOAT Analytical Model

[![Google Sheets](https://img.shields.io/badge/Google_Sheets-View_Live_Model-success?style=for-the-badge&logo=googlesheets)](https://docs.google.com/spreadsheets/d/1wCGeV-194F5a9CaFOrZiJQmLf2ba2pUVidcrwqGBIYE/edit?usp=sharing)

## Project Overview
This project is a data-driven approach to the "Greatest of All Time" debate in the NFL. 
Unlike traditional rankings, this model uses **era-adjustment** and **z-score normalization** to compare players across different statistical environments.

## The Four-Pillar Model
- **Greatest:** Career accolades and historical legacy.
- **Best:** Peak efficiency and rate-based metrics (ANY/A+).
- **Dominant:** Performance relative to league averages per era.
- **Clutch:** Postseason elevation and "Big Game" performance.

## Skills Demonstrated
- **Statistical Modeling:** Z-Score Normalization, Weighted Scoring.
- **Advanced Excel:** XLOOKUP, Multi-sheet Architecture, Conditional Logic.
- **Data Visualization:** Custom Dashboarding and Radar Charts.

## How to Use
The spreadsheet is organized into a Presentation Layer (Dashboard/Rankings) and 
a Calculation Layer. Hidden sheets contain the raw normalization math.
